%f     The current file name in the active panel.

%x     The extension of the current file name.

%b     The current file name without extension.

%d     The directory of the active panel.

%F     The current file in the inactive panel.

%D     The directory of the inactive panel.

%t     The currently tagged files.

%T     The tagged files in the inactive panel.

%u and %U
       Similar to %t and %T, but after substitution the files are untagged.

%s and %S
       Equivalent to %t and %T if there are tagged files, otherwise equivalent to %f and %F.
